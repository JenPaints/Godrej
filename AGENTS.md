## Project Summary
A premium real estate landing page for "Godrej Yelahanka", a residential project in Bangalore. The page features project overview, gallery, amenities, floor plans, pricing, and a lead capture system.

## Tech Stack
- HTML5
- CSS3 (with Custom Properties)
- JavaScript (Vanilla)
- Google Fonts (Inter)
- Font Awesome (Icons)

## Architecture
- Simple flat structure: `index.html`, `styles.css`, `script.js`.
- Assets directory for project images.
- Responsive design with mobile-first considerations for key elements.

## User Preferences
- Clean, professional, and luxury aesthetic for real estate.
- Use of blue/white color palette consistent with corporate branding.

## Project Guidelines
- Maintain glassmorphism effects where appropriate.
- Ensure smooth animations and transitions.
- Validate all form inputs before submission.
- Keep the code well-organized and avoid unnecessary comments unless requested.

## Common Patterns
- Scroll-based animations using Intersection Observer.
- Modal-based lead capture forms.
- Tabbed content for amenities or other categorized info.
- Custom slider for testimonials and gallery.
